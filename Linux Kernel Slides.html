Linux Kernel

Giới thiệu về nhân hệ điều hành Linux

Linux Kernel là gì?

  • Là thành phần cốt lõi của hệ điều hành Linux
  • Quản lý tài nguyên phần cứng và phần mềm
  • Cầu nối giữa phần cứng và ứng dụng người dùng
  • Mã nguồn mở, được phát triển bởi cộng đồng

Chức năng chính

  • Quản lý bộ nhớ (Memory Management)
  • Lập lịch tiến trình (Process Scheduling)
  • Quản lý thiết bị (Device Management)
  • Hệ thống tập tin (File Systems)
  • Quản lý mạng (Network Stack)

Kernel Space vs User Space

  • Kernel Space: Quyền truy cập hoàn toàn vào phần cứng
  • User Space: Môi trường cho ứng dụng người dùng
  • System Calls: Giao diện giữa User Space và Kernel Space

Kernel Modules

  • Mở rộng chức năng kernel không cần biên dịch lại
  • Có thể tải và gỡ bỏ động
  • Ví dụ: Device drivers, File systems, Network protocols

Process Management

  • Tạo và hủy tiến trình
  • Lập lịch CPU
  • Inter-Process Communication (IPC)
  • Thread management

Memory Management

  • Virtual Memory
  • Page Tables
  • Memory Allocation
  • Swap Space Management

File System

  • Virtual File System (VFS)
  • Hỗ trợ nhiều loại file system
  • Buffer Cache
  • I/O Scheduling

Security

  • Access Control
  • Security Modules (SELinux, AppArmor)
  • Capabilities
  • Cryptographic API
