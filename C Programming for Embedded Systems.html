  • Pointer và Memory Management

1. Pointer và Memory Management

  • Pointer to specific memory addresses
  • Memory mapped I/O
  • Direct register access

volatile uint32_t* port = (uint32_t*)0x40020000;
*port = 0x00000001; // Set pin high

2. Volatile & Const Qualifiers

  • Volatile: Prevent optimization
  • Const: Read-only data in ROM

volatile uint8_t status_reg;  // Hardware register
const uint8_t VERSION = 0x01; // ROM constant

3. Interrupt Handling

  • ISR (Interrupt Service Routines)
  • Interrupt vectors
  • Priority levels

void __interrupt__ timer0_isr(void) {
  if(PIR1bits.TMR0IF) {
      // Handle interrupt
      PIR1bits.TMR0IF = 0;
  }
}

4. Bit Manipulation

  • Bitwise operations
  • Bit fields
  • Bit masks

// Set bit 3
PORTA |= (1 << 3);
// Clear bit 3
PORTA &= ~(1 << 3);
// Toggle bit 3
PORTA ^= (1 << 3);

5. Structure và Union trong Embedded

  • Bit fields trong struct
  • Packed structures
  • Memory alignment

typedef struct {
  unsigned int flag1 : 1;
  unsigned int flag2 : 1;
  unsigned int data  : 6;
} __attribute__((packed)) StatusReg;

6. Real-time Programming

  • Task scheduling
  • Timer management
  • Event handling

void main(void) {
  while(1) {
      if(timer_expired()) {
          process_tasks();
          reset_timer();
      }
  }
}

Best Practices

  • Sử dụng const khi có thể
  • Tránh dynamic memory allocation
  • ISR phải ngắn gọn và nhanh
  • Sử dụng volatile đúng cách
  • Kiểm tra boundary conditions
  • Code phải portable và maintainable
